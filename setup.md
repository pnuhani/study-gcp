# Quick Start Guide

## Installation

1. Clone repository:
```bash
git clone <repo name(study-gcp)>
cd study-gcp
```

2. Navigate to frontend and install dependencies:
```bash
cd frontend
npm install
```

3. Start development server:
```bash
npm run dev
```
